FONTLOG
Le Patin helvète
================


This file provides detailed information on Le Patin helvète font software.
This information should be distributed along with the Le Patin helvète fonts and any derivative works.

Basic Font Information
----------------------

Le Patin helvète is a re-interpretation of Nimbus Sans, to which we added clogs.


For more detailed info, see http://ospublish.constantvzw.org


Le Patin helvète currently provides the following Unicode coverage:
Basic Latin: 95/95 
Latin-1 Supplement: 93/96
Latin Extended-A: 128/128
Latin Extended-B: 7/208
Cyrillic: 197/256
Latin Extended Additional 90/256
General Punctuation: 16/107
